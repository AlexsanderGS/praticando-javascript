# 🚀 Praticando JavaScript

Repositório criado para registrar minha evolução nos estudos de **JavaScript**, contendo exercícios práticos, desafios e experimentações realizadas durante minha formação em **Desenvolvimento Front-End**.

O objetivo deste projeto é consolidar os fundamentos da linguagem através da prática constante, explorando conceitos essenciais que servem como base para frameworks modernos como **React**.

---

## 📚 Sobre o Projeto

Este repositório reúne exercícios desenvolvidos durante meus estudos de JavaScript, abordando desde conceitos básicos até funcionalidades mais modernas da linguagem (ES6+).

Os códigos foram escritos com foco em:

- ✅ Fixação da lógica de programação
- ✅ Entendimento da sintaxe do JavaScript
- ✅ Boas práticas iniciais
- ✅ Aprendizado progressivo através da prática

---

## 🧠 Conteúdos Praticados

### 🔹 Fundamentos
- Variáveis (`let`, `const`)
- Tipos de dados
- Operadores
- Condicionais (`if`, `else`, operador ternário)
- Laços de repetição (`for`, `for...of`, `for...in`)
- Funções

### 🔹 Arrays
- Manipulação de arrays
- Métodos:
  - `map`
  - `filter`
  - `forEach`
  - `push`
- Destructuring em arrays
- Spread e Rest operator

### 🔹 Objetos
- Criação de objetos
- Propriedades e métodos
- Acesso a atributos
- Métodos nativos:
  - `Object.keys()`
  - `Object.values()`
  - `Object.entries()`

### 🔹 JavaScript Moderno (ES6+)
- Template Strings
- Arrow Functions
- Destructuring
- Spread & Rest
- Módulos (`import` / `export`)

### 🔹 Datas
- Objeto `Date`
- Formatação com `toLocaleDateString`

### 🔹 Node.js (ambiente de execução)
- Uso do terminal
- Entrada de dados com `readline`
- Organização em módulos
